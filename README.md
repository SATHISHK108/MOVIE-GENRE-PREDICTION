# MOVIE-GENRE-PREDICTION
Movie genre prediction is to identify the genre of the movie from dialogues.

STEP 1:
  Reading the CSV file, and do Exploratory Data Analysis(EDA) 
  and Visualizing the genre count in the data.

STEP 2:
  Clean the text column using nltk and create corpus
  
STEP 3:
  Visualize the words related to genre using wordcloud
  
STEP 4:
  Using countvectorizer to fit_transform the corpus
  and doing train_test_split.
  
STEP 5:
  Using Navie Bayes classifier and hypertunning the alpha value to get better prediction score.
  
 
        *** Finally arrived with an accracy of 91.34% with alpha value as 0.1 ***
